• 使用babel-loader转换ES6代码
  • babel-loader用法
    安装 babel-loader
    $ npm install babel-loader --save-dev

    安装 babel-core
    $ npm install babel-core --save-dev

    安装转码规则
    $ npm install babel-preset-es2015

    配置webpack.config.js
    参考格式
    {
        test: /\.js$/,
        loader: 'babel?presets[]=es2015'
    }

    执行webpack命令
    $ webpack
