• babel-loader用法
    1. 安装 babel-loader
    $ npm install babel-loader --save-dev

    2. 安装 babel-core
    $ npm install babel-core --save-dev

    3. 安装转码规则
    $ npm install babel-preset-es2015

    4. 配置webpack.config.js
    参考格式:
    rules: [
        {
            test: /\.js$/,
            exclude: /(node_modules | bower_components)/,
            use: {
                loader: 'babel-loader',
                options: {
                    presets: ['es2015']
                }
            }
        }
    ]

    5. 执行webpack命令
    $ webpack
